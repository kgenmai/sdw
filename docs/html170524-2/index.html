2017年度第1回の英検より3級・準2級にライティングテストが導入されます。特別な対策が必要になります。

英検を取得しておくと学校内での単位認定や内部進学の際にも役立ちます。10月英検に向けてこの機会に学習をスタートさせましょう。

日本人講師がわかりやすく、きめ細やかに指導する“英検に特化した”コースです。
シェーンの日本人講師はおもに、英検準1級取得の人材を採用。文科省の指導要領を加味しながら、listening,speaking,reading,writingの4スキルの向上を図れるよう指導しています。

中１で準２級を目指しています！

幼稚園からネイティブ講師のレッスンを始め小2から日本人講師も併せて受講。今年１月の英検で見事３級合格。今もネイティブ講師を週1回、英検準2級対策を週1回受講中です。「英語をやって良かったと思えることは英語の音楽を聴いていると歌詞が3回くらいでわかったことです」。ディズニーの映画を観るのも大好き。英語字幕にして耳で聴きながら英語を目で確認もしています。将来はスペイン語やイタリア語を英語で習得するのが目標。英語だけではなく数か国語の通訳を目指しています。

担当の日本人講師からコメント

小2から日本人講師のレッスンを併せて受講しているので順当な進捗です。準2級のレッスンではより広い分野の英語に触れ、抽象的な語彙の習得も求められます。普段からニュースを見たり読書をしたりと、知識を増やすことを心がけましょう。もちろんレッスンの復習は欠かさずに。自分の言葉として発信できる英語の習得に向けて、作文や発話の練習にも意識的に取り組みましょう。
